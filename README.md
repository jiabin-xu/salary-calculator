# 薪资计算器

## 首页

- 选择城市
- 选择城市后可以带入城市的社保和公积金基数
- 社保基数选择：最低基数、按照工资、自定义
- 公积金基数选择：最低基数、按照工资、自定义
- 选择公积金缴纳比例
- 个税专项扣除
- 年终奖多少月工资，发放日期，个税计算方式

## 薪资分布

- 社保缴纳详情
- 公积金缴纳详情
- 个税抵扣详情
- 全年 12 个月的税后工资
- 税后薪资
- 税前薪资
- 年薪

## 最简洁的路径

- 输入税前薪资
- 选择城市
- 默认社保基数为跟随工资
- 默认公积金基数为跟随工资
- 默认公积金缴纳比例为 12%
- 默认个税专项扣除为 0
- 默认年终奖发放月份为 12 个月
- 默认年终奖个税计算方式为 按月换算
